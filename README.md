# PEAK AGENCY

A premium, modern, responsive website for Peak Agency — a Cameroon-based creative agency with global ambition.

## Brand

**Slogan:** "Des supports qui élèvent votre image."

**Personality:** Premium, Bold, Strategic, Modern, Minimal, Professional, Creative, Global

## Tech Stack

- **Next.js 14** — React framework with App Router
- **TypeScript** — Type-safe development
- **Tailwind CSS** — Utility-first styling
- **Framer Motion** — Smooth animations and scroll reveals
- **Lucide React** — Clean, modern icons

## Features

- Smooth scroll navigation
- Scroll-triggered animations (Framer Motion)
- Premium hover effects on cards and buttons
- Fully responsive (mobile-first)
- SEO metadata
- Accessible HTML structure
- Dark premium aesthetic with electric blue/purple accents

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Sections

6. **Why Peak** — 4 differentiators with hover effects
7. **CTA** — Bold call-to-action section
